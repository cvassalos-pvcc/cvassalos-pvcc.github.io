Briefly describe or define:

  1. information security - protecting data from those who don't have permission to access it
  2. authentication - verifying that a user has permission to access a particular device, whether it be a local machine, or remote
  3. dual authentication - after entering log-in credentials, a second device will receive a temporary code that must be entered to gain access
  4. social engineering - the process of using people to get the information you want
  5. virus - a program that infects a computer and spreads to others, similar to a biological virus
  6. Trojan horse - a program that does a legitimate computational task, but also contains malicious code to infect a computer unbeknownst to the user
  7. ransomware - malicious code, usually downloaded by email, that will strongly encrypt files on the computer and then demand payment from the user for the files to be decrypted
  8. phishing - not a direct attack on a computer, typically it is someone trying to get a user's personal information, financial information or social security number, via an email that appears to be from a legitimate organization
  9. White Hat - security experts that are trying to keep users safe from malicious attacks
  10. Black Hat - security experts that attempt to exploit users through malicious attacks
  11. encryption/decryption - encryption is the process of using an algorithm to convert information into a representation that cannot be understood or utilized by anyone until it is decrypted, which is the reverse, using an algorithm to convert the ciphertext back into plain-text
  12. cryptography - the science of "secret writing"
  13. symmetric encryption algorithim - requires the use of a secret key known both to sender and receiver, the sender encrypts the file with this secret key, and the reciever user uses the same secret key to decrypt the file
  14. Caesar cipher - also called a shift cipher, it involves shifting character in a message to another character some fixed distance further along in the alphabet/li>
  15. block cipher - a method of encrypting data in blocks to produce ciphertext using a cryptographic key and algorithm
  16. Data Encryption Standard (DES) - developed by IBM in the 70's, it is a certified as an international standard used for encrypting data
  17. public key system - an asymmetrical encryption algorithm, the encryption key for a messages goes to everyone, but only the receiver of the message gets the decryption key
  18. embedded computers - computational devices such as chips, processors, or computers that are embedded within another system

Briefly describe how a text file can be hidden in a steganographic image:

You can hide a text file in an image using the RGB values. Each color, red, green, and blue, has 8 bits allocated to them to manipulate the intensity of the color. A user can change the last bit in each color to convey a message, and because it is the smallest bit it will change the color by 1/256th, which is not noticeable at all.

List six ways to protect yourself from most cyber attacks

  1. Be sure your computer has up-to-date antivirus software.
  2. Be sure your computer has up-to-date firewall software.
  3. Be sure your computer has up-to-date antispyware.
  4. Set your browswer to use a pop-up blocker.
  5. Always install the latest security patches or updates to your operating system.
  6. Do not send personal or financial information in response to any email.
