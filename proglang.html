Programming Languages

machine language the language understand by the computer, and not by humans
debugging the process of looking for errors in computer programs
assembler computer program that takes assembly language and converts it for the computer processor to understand
assembly language an old and low level programming language
translator general term for a conmputer program that converts a programming language to another language
compiler a computer program that converts an entire file of a high level language to a low level language
interpreter a computer program that converts a high level language to a low level language line by line, pointing out errors along the way
cross-platform software software that is designed to work on several platforms
natural language the language we speak and read in everyday life
machine independence programming language that can operate on any computer
high-level programming language languages that humans can understand, and need to be converted for the computer to understand
object-oriented programming (OOP) a computer programming model that organizes software design around data, or objects, rather than functions and logic
scripting language interpreted at runtime instead of compiled, it is a programming language that is used to manipulate, customize, and automate the facilities of an existing system
data types
integer Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length
float Float, or "floating point number" is a number, positive or negative, containing one or more decimals
string data type that holds text
primitive data type these data structures contain simplified data values and serve as the foundation for manipulating data
data structure a way of organizing data so that it can be accessed more efficiently depending upon the situation. Data Structures are fundamentals of any programming language around which a program is built
Python collection data structures
list used to store multiple items in a single variable, are mutable, and ordered
tuple used to store multiple items in a single variable, are not mutable, and ordered
set used to store multiple items in a single variable, are mutable, and unordered
dictionary Dictionaries are used to store data values in key:value pairs, a dictionary is a collection which is ordered, changeable and do not allow duplicates.
function A function is a block of code which only runs when it is called, you can pass data, known as parameters, into a function, and a function can return data as a result.
local variable A variable created inside a function belongs to the local scope of that function, and can only be used inside that function
global variable Variables that are created outside of a function (as in all of the examples above) are known as global variables. Global variables can be used by everyone, both inside of functions and outside.
source program an original computer program written by a programmer that is converted into the equivalent object program, written in machine language, by the compiler or assembler
object program a computer program translated from the equivalent source program into machine language by the compiler or assembler
