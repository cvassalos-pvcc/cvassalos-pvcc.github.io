All About Github

  • What is a version control system?
  • It is a system in software engineering used to track and manage all changes made to code.
  • What is Git?
  • Git is a free and open source version control system.
  • What is GitHub?
  • GitHub is the website that hosts the version control software.
  • What is the difference between Git and GitHub?
  • Git is the software that does version control, GitHub hosts the software online and allows the users to manage their Git repositories.
  • Who started GitHub and how was it started?
  • GitHub was created by three individuals named Tom Preston-Werner, Chris Wanstrath, and PJ Hyett. It begin as a weekend side project to assist them in their main coding projects
  • What company owns it now?
  • Microsoft now owns GitHub.
  • How much does a GitHub account cost?
  • A GitHub account is free, but for more advanced features there are multiple price plans.
  • What is the octocat?
  • Octocat is the mascot for GitHub. It is half cat and half octopus.

Git/GitHub Terms

repository
It is the .git folder that tracks all the changes made to the code.
commit
A snapshot of you the project you are working on
fork
Makes a copy of code that can then be worked on without changing the original
push
Uploads a local repository to a remote repository
pull requests
Tells other coders what changes you have made to code that has been pushed to a repository
workflows
The workflow is how you go about using Git to encourage productivity
issues
