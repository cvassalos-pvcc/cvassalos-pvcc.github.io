Networking and the Internet
Network Classifications
  • LAN - "local area network", a network connecting computers and devices in a relatively close proximity
  • MAN - a medium distance computer network, typically the size of several blocks up to the size of an entire city, MAN stands for "metropolitan area network"
  • WLAN - a local computer network, made up of two or more devices, that is connected wirelessly
Network Models
  • Bus Network - a network topology where nodes are all directly connected to a network link called a bus
  • Star Network - a network topology where all devices are connected to one central hub
Protocols
  • CSMA/CD Ethernet - Carrier-Sense Multiple Access with Collision Detection, it is a method for transfering data over a network by defering transfer until the network is available
  • CSMA/CA Wifi - Carrier-Sense Multiple Access with Collision Avoidance,
Combining Networks
  • repeater - a device that amplifies and sends a signal forward to strengthen LAN network
  • bridge - like a repeater but "smarter", it sends a signal forward based on if the destination is on another "node" or network
  • switch - connects devices on a network by receiving data and forwarding it to the correct port
  • router - connects two or more networks
  • gateway - a piece of networking hardware or software used in telecommunications networks that allows data to flow from one discrete network to another
Client/Server Model
  • P2P: peer-to-peer - Peer-to-peer computing or networking is a distributed application architecture that partitions tasks or workloads between peers
  • distributed system - environment where multiple computers or devices are working on a variety of tasks and components, all spread across a network
  • cloud computing - the practice of using a network of remote servers hosted on the internet to store, manage, and process data, rather than a local server or a personal computer
Internet Addressing
  • Tier 1 - a network that can reach every other network on the Internet solely via settlement-free interconnection
  • Tier 2 - a network that peers for free with some networks, but still purchases IP transit or pays for peering to reach at least some portion of the Internet
  • Tier 3 - a network that solely purchases transit/peering from other networks to participate in the Internet
IP Address
  • ICANN - Internet Corporation for Assigned Names and Numbers, an American multistakeholder group and nonprofit organization responsible for coordinating the maintenance and procedures of several databases related to the namespaces and numerical spaces of the Internet
  • DNS - Domain Name System, a naming system for computers, services, and other resources in the Internet or other Internet Protocol networks
  • TLD - Top Level Domain, one of the domains at the highest level in the hierarchical Domain Name System of the Internet after the root domain.
  • Country-Code TLDs - an Internet top-level domain generally used or reserved for a country, sovereign state, or dependent territory identified with a country code.
Internet Protocols
  • FTP - File Transfer Protocol, a standard communication protocol used for the transfer of computer files from a server to a client on a computer network
  • HTTP - Hyptext Transfer Protocol, an application layer protocol in the Internet protocol suite model for information systems
  • SMTP - Simple Mail Transfer Protocol - an Internet standard communication protocol for electronic mail transmission
World Wide Web
  • URL - Uniform Resource Locator, a reference to a resource that specifies its location on a computer network and a mechanism for retrieving it
  • client-side activities - when a web application takes place on the user's device
  • server-side activities - when a web application takes place on a server, instead of the user's device
Internet Four-Layer Software
  • Application layer - the rules for implementing the end-user services provided by a network
  • Transport layer - this layer provides end-to-end communication services for devices
  • Network layer - the network layer is responsible for packet forwarding including routing through intermediate routers
  • Link layer - this layer is the protocol layer that transfers data between nodes on a network segment across the physical layer
Internet is:
  • packet-switching network - sending large data composed of small packets from one place on the internet to another
  • fault tolerant - there are many backups to keep a large scale failure of the interent, disruptions should only happen on a small scale
  • redundant
Cybersecurity
  • malware - software that is specifically designed to disrupt, damage, or gain unauthorized access to a computer system
  • virus - a program that infects a computer and spreads to others, similar to a biological virus
  • worm - a type of malware that can replicate itself without the user doing anything
  • trojan horse - a program that does a legitimate computational task, but also contains malicious code to infect a computer unbeknownst to the user
  • spyware - malware designed to record what happens on the infected device
  • phishing - a social hacking technique where fraudulent emails are sent to get secure user information
  • DDoS attack - Distributed Denial-of-Service Attack, a cyber attack where service is disrupted by an overwhelming amount of connecxtions are attempted to be made
Protection
  • firewall - hardware and software put in place to prevent unauthorized access to a company's or individual's private data
  • spam filter - an email filter that attempts to keep junk, phishing, and spam emails out of the users inbox
  • proxy server - a system or router that provides a gateway between users and the internet
  • antivirus software - software designed to scan compters for viruses, as well as any incoming files to make sure they don't contain malware
  • cryptography - he process of hiding or coding information so that only the person a message was intended for can read it
  • public key encryption - an asymmetrical encryption algorithm, the encryption key for a messages goes to everyone, but only the receiver of the message gets the decryption key
